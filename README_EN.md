## How to run locally

1. Clone the repository:
```bash
git clone https://github.com/your-username/immersive-form-en.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run interface_formulaire_en.py
```
